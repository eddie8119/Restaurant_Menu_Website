<h1> 專案介紹 </h1>
利用Node.js環境 + Express框架，做一個餐廳資訊網站

<h1> 操作功能 </h1>
<ul>
  <li>點擊餐廳資訊卡可以看到個別餐廳更多資訊Coffee</li>
  <li>使用餐廳搜索關鍵字或是類別，搜尋餐廳</li>  
</ul>

<h1> 專案使用工具 </h1>
<ul>
  <li>Node.js -執行環境</li>
  <li>express:4.16.4 -框架</li>
  <li>express-handlebars:3.0.0 -樣板引擎</li>
  <li>bootstrap:4.3.1</li>  
</ul>
